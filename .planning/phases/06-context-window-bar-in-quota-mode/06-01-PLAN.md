| Boundary | Description |
|----------|-------------|
| user config file → load_config | `~/.claude/claude-statusbar.json` is user-editable; a garbage `show_context` value must not crash the render. |
| CLI arg → set_value | `cs config set show_context <value>` accepts arbitrary strings. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-06-01 | Tampering | load_config(show_context) | low | mitigate | `_to_bool` coerces any raw value; load_config already returns default StatusbarConfig on OSError/JSONDecodeError, so a corrupt file falls back to show_context=True. |
| T-06-02 | Denial of Service | set_value(show_context) | low | accept | Boolean path never raises on bad input (`_to_bool` maps unknowns to False); no crash surface. |
| T-06-SC | Tampering | package installs | low | accept | No npm/pip/cargo installs in this plan — stdlib-only field addition. No legitimacy gate required. |
</threat_model>
