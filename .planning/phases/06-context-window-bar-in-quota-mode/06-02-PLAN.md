<objective>
Make context render as a per-style bar/segment in QUOTA mode, gated by a `show_context` kwarg
(default False so existing callers/tests are unaffected). This is the visual heart of the phase:
CTX-01 (bar segment per style, not a model suffix) and CTX-02 (severity band 70/85). The segment
reuses each style's existing no-quota context rendering so quota and no-quota modes look identical
(LOCKED "Per-style segment" + "Severity band" decisions in 06-CONTEXT.md).

Purpose: Today, in quota mode, context is only a `(used/size)` suffix + a tint on the model name.
This plan adds the real bar/pill/mini-bar between the 7d segment and the model, and neutralizes the
model (the bar now carries context severity). Core/preview wiring (drop the suffix, pass the flag)
lands in plan 06-03; this plan makes the renderers ready.

Output: `show_context` kwarg on format_status_line + the three style renderers; the quota-mode ctx
segment placed between 7d and model in each style's idiom; a mirrored render test.
</objective>
