- `theme=str(raw.get("theme", DEFAULT_THEME))`  (line ~143 — covered by the constant)

Why both matter: a fresh install with NO file returns `StatusbarConfig()` (Location A). A user with a PARTIAL config file reaches the `raw.get(key, default)` fallback (Location B) for any key they never set. If A and B disagree, behavior differs depending on whether some unrelated key happens to be present in the file — a silent, hard-to-spot bug.

For `theme`, both paths read the `DEFAULT_THEME` constant, so flipping the constant on line ~19 covers Location A AND Location B for theme. VERIFY this — do not add a hardcoded `"tokyo-night"` anywhere the constant is expected.
